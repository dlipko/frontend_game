# frontend_game

Для запуска на ноде:  
создайте в главной деректории файл package.json  
пример файла:
```
{
  "name": "first",
  "version": "1.0.0",
  "description": "",
  "main": "app.js",
  "keywords": [],
  "author": "",
  "license": "ISC",

  "scripts": {
    "start": "node ."
  },
  "dependencies": {
    "micro": "latest"
  }
}
```

Запустите команду

```
node .
```

Для запуска на now.sh
```
now
```
